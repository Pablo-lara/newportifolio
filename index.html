Projetos

Cadastro de pessoas github

Esse projeto usa principalmente REACT e JavaScript e integra a API em node para cadastrar pessoas no banco de dados. O sistema permite cadastrar e excluir pessoas além de exibir todos os cadastros do banco de dados na interface do site.

Projeto Previsão do Tempo site github

Esse projeto foi desenvolvimento usando HTML,CSS e JavaScrpit, integrando uma API externa que fornece dados sobre clima. Nesse sistema você coloca o nome de uma cidade e os dados sobre clima dela serão puxados e apareceram na tela, é possivel ver dados como: temperatura, umidade e velocidade do vento.

API cadastro nodeJS github

Esse projeto é uma API própria feita usando nodeJS integrando um banco de dados. Basicamente esse sistema permite cadastrar, remover e editar pessoas no banco de dados, o banco de dados em questão é o MongoDB.

TodoList site github

Nesse projeto, desenvolvi um TodoList usando principalmente JavaScript puro, além também de usar algumas ferramentas de HTML e CSS. O projeto conta com funcionalidades de adição, edição e remoção de tarefas, além de uma interface agradável de usar.

SnakeGame site github

Relogio digital site github
